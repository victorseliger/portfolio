# Portfolio — Victor Seliger

> 🌐 **Language:** **English** · [Português](../README.md)

> **Engineering at the boundary between business rules and data.**
> I solve manual business pains with **automation + well-crafted specifications** — from financial
> reconciliation pipelines to platforms and AI applied in production.

This repository is a public portfolio: it gathers, as **anonymized case studies**, what I've built
throughout my career using data engineering, automation and **Claude Code / applied AI**.

Due to confidentiality (NDA), **no client, company, proprietary code or production data is exposed**.
Each case uses only an industry hint (e.g. "energy multinational", "national leader in batteries")
and focuses on the **transferable technical pattern** — the problem, the approach, the architecture
and the result.
