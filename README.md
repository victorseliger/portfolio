# Portfólio — Victor Seliger

> **Engenharia na fronteira entre regra de negócio e dado.**
> Resolvo dores manuais de negócio com **automação + especificação bem feita** — de pipelines de
> conciliação financeira a plataformas e IA aplicada em produção.

Este repositório é um portfólio público: reúne, em formato de **case study anonimizado**, o que
construí ao longo da carreira usando engenharia de dados, automação e **Claude Code / IA aplicada**.

Por confidencialidade (NDA), **nenhum cliente, empresa, código proprietário ou dado produtivo é
exposto**. Cada case usa apenas dica de setor (ex.: "multinacional de energia", "líder nacional em
baterias") e foca no **padrão técnico transferível** — o problema, a abordagem, a arquitetura e o
resultado.
